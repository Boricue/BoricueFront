EJECUCION:

Para ejecutar el proyecto como desarrollador necesitas tener instalado NODE.JS en tu dispositivo, una vez hecho esto debes presionar ctrl+ñ o abrir la terminal
y digitar en este orden:

npm i
npm run dev
